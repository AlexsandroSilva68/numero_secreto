<h1>Número Secreto</h1>

<p>Este projeto foi utilizado para aprendizado, onde o projeto foi disponibilizado para fazer um curso, onde explorei o uso das ferramentas Git e GitHub para controle de versão e gerenciamento de repositórios, além de trabalhar com o terminal para realizar operações básicas e avançadas.</p>

<h2>Objetivo do Projeto</h2>

<ul>
    <li>Compreender como importar projetos existentes para o Git.</li>
    <li>Aprender a usar o terminal para interagir com o Git.</li>
    <li>Praticar os comandos essenciais do Git, como <code>git add</code>, <code>git commit</code>, <code>git push</code>, <code>git pull</code>, entre outros.</li>
    <li>Familiarizar-me com o processo de subir projetos no GitHub e colaborar em repositórios remotos.</li>
</ul>

<h2>Tecnologias Utilizadas</h2>

<ul>
    <li><strong>Git:</strong> Para controle de versão e gerenciamento de repositórios.</li>
    <li><strong>GitHub:</strong> Para hospedar o repositório e colaborar em projetos remotos.</li>
    <li><strong>Terminal (onde utilizei o próprio terminal da IDE):</strong> Para execução de comandos Git e navegação no sistema de arquivos.</li>
    <li><strong>IDE:</strong> Visual Studio Code para edição de código e configuração do ambiente de desenvolvimento.</li>
</ul>

<h2>Comandos Aprendidos</h2>

<ul>
    <li><code>git init</code>: Inicializa um novo repositório Git.</li>
    <li><code>git clone</code>: Clona um repositório remoto para o seu ambiente local.</li>
    <li><code>git status</code>: Verifica o status dos arquivos no repositório.</li>
    <li><code>git add</code>: Adiciona arquivos modificados ao staging area.</li>
    <li><code>git commit -m "mensagem"</code>: Cria um commit com as mudanças feitas.</li>
    <li><code>git push</code>: Envia as mudanças commitadas para o repositório remoto no GitHub.</li>
    <li><code>git pull</code>: Atualiza o repositório local com as mudanças do repositório remoto.</li>
</ul>

    <li>Como o GitHub pode ser usado para compartilhar projetos, colaborar com outros desenvolvedores, e manter um portfólio online.</li>
</ul>
